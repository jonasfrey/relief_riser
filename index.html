    2. Image editor

    Higher resolution = more detail, larger STL, slower preview.

    Gradient frame

    Black-to-transparent overlay along each edge, fading to no effect over the given thickness. Mixes into the heightmap to chamfer the top of the relief. Set 0 to disable an edge.

    Edge blending

    Cross-fades opposing edges of the heightmap into each other so the wrap-around seam in cylindrical / custom-profile shapes (and tile boundaries) blends away. Blend width is shared between the two axes.

    3. Shape parameters (mm)

    No target STL loaded

    The image wraps around the STL's outer surface using cylindrical projection along its Z axis. The STL should enclose its Z axis (e.g. a vase shape — centered on its own XY bounding box). The W slider is ignored: circumference and height come from the STL bounding box. With auto-fit on, "Repeat X" is derived so tiles preserve the image aspect ratio while staying integer (= seamless around the circumference); only "Repeat Y" remains user-controlled.

    X and Y define the inner cavity dimensions. Wall thickness grows outward, so the outer ellipse = inner + thickness. The image wraps once around the outer face; each greyscale pixel pushes its outer vertex further from the center. Bottom hole is a percentage of the inner X/Y (50% = hole is half the inner size, 100% = fully open).

    Scale X and Y define the inner cavity dimensions. Wall thickness grows outward, so outer dimensions = inner + 2 × thickness. The image wraps once around all four outer faces; each greyscale pixel pushes its outer vertex away from the face.

    No profile loaded
    Band: auto · click profile to pick manually

    DXF should be a single closed loop of LINE / ARC / CIRCLE / ELLIPSE / LWPOLYLINE entities. Profile X = radial distance from rotation (Z) axis; Profile Y = axial coord. The "outer band" (right-facing arc whose X is within this fraction of max X) gets the relief offset.
